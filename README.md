# MMT Landing Page

This is the jekyll project for Metromedia Technologies landing page. It uses Webpack, Babel, Sass, and Jekyll.

## Installation
Requires Node and NPM

Install dependencies
```
npm install
```

Run dev server
```
npm start
```

Build production
```
npm run build
```
